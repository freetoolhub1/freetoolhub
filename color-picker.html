What is a Color Picker?

A color picker lets you select any colour and instantly get its code in HEX, RGB, and HSL formats — the three colour systems used in web design and development.

HEX vs RGB vs HSL

  • HEX (#a78bfa) — used in HTML and CSS. The most common format for web development.
  • RGB (167,139,250) — Red, Green, Blue values 0-255. Used in CSS and design software.
  • HSL (258,90%,76%) — Hue, Saturation, Lightness. Easiest format for humans to understand and adjust.
