What Makes a Password Strong AND Memorable?

The best passwords are ones you can remember without writing them down — but that are still impossible to guess. A passphrase like "coffee-lamp-seven-sunrise" is 26 characters long, contains no dictionary attack vulnerability, and is far easier to remember than "xK#9mP2q".

Password Types Explained

  • Passphrase — 4+ random words joined by hyphens. Long, strong, memorable. Recommended by NIST security guidelines.
  • Pattern-based — uses a memorable structure like Word+Number+Symbol+Word. Shorter but strong.
  • Fully random — maximum security. Use a password manager to remember it.

Password Security Rules

  • Never reuse passwords across sites
  • Longer beats complex — 20 simple characters beats 10 complex ones
  • Use a different password for every account
  • Store passwords in a password manager like Bitwarden (free) or 1Password
Frequently Asked Questions
What is a passphrase?+
A passphrase is 4+ random words joined together. 'correct-horse-battery-staple' is 28 characters, extremely strong, and memorable.
Are these passwords secure?+
Yes. Passphrases with 4+ random words have quadrillions of possible combinations. Pattern-based passwords are also very strong with 12+ characters.
Should I use these exact passwords?+
Use them as inspiration or use them directly. The key is to choose one you can remember and use it only on one account.
What password manager should I use?+
Bitwarden is free and open source. 1Password and Dashlane are excellent paid options. All store passwords securely so you only remember one master password.
