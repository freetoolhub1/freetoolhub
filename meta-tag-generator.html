Essential Meta Tags for SEO

  • title — shown in browser tabs and search results. Under 60 chars.
  • description — the search result snippet. Under 155 chars.
  • canonical — tells Google the preferred URL version.
  • og:image — the image shown when shared on social media.
Frequently Asked Questions
What meta tags does it generate?+
Title, description, canonical, Open Graph (Facebook, LinkedIn, WhatsApp) and Twitter Card tags.
What is the optimal title length?+
Under 60 characters. Google truncates longer titles in search results.
Do meta tags affect SEO rankings?+
The title tag directly affects rankings. Meta description affects click-through rate but not rankings directly.
What is Open Graph?+
OG tags control how your page appears when shared on social media — the title, description and image shown in the preview.
